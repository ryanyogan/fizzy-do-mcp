# Introduction to Fizzy MCP

Fizzy MCP is a **Model Context Protocol (MCP)** server that enables AI assistants to interact with [Fizzy](https://app.fizzy.do), the AI-powered task management platform built for modern teams.

## What is MCP?

The [Model Context Protocol](https://modelcontextprotocol.io) is an open standard developed by Anthropic that allows AI applications to securely connect to external data sources and tools. It provides a standardized way for AI assistants to:

- **Access data** from external systems
- **Execute actions** through well-defined tools
- **Maintain context** across conversations

## Features

### 40 MCP Tools

Fizzy MCP provides comprehensive API coverage across 7 categories:

| Category | Tools | Description |
|----------|-------|-------------|
| Identity | 2 | Account discovery and settings |
| Boards | 7 | Board CRUD and publishing |
| Cards | 18 | Card lifecycle, assignments, tags |
| Comments | 5 | Discussion threads on cards |
| Columns | 5 | Workflow stage management |
| Tags | 1 | Label management |
| Users | 2 | Team member information |

### Auto-Configuration

After authentication, Fizzy MCP can automatically detect and configure:

- **Claude Desktop** - Anthropic's official desktop app
- **Cursor** - AI-powered code editor
- **OpenCode** - Open-source AI coding assistant

### Type-Safe Client

Built with TypeScript and Zod validation:

```typescript
import { FizzyClient } from '@fizzy-mcp/client';

const client = new FizzyClient({
  accessToken: 'your-token',
  accountSlug: '/your-org',
});

// Fully typed responses
const result = await client.cards.list({ board_ids: ['abc123'] });
if (result.ok) {
  console.log(`Found ${result.value.length} cards`);
}
```

### Resilient API Calls

Automatic retry logic with exponential backoff:

- Retries on network errors and 5xx responses
- Respects rate limit headers
- Configurable retry count and delays

## Prerequisites

Before getting started, you'll need:

1. **A Fizzy account** - Sign up at [app.fizzy.do](https://app.fizzy.do)
2. **Node.js 20+** - Required for running the MCP server
3. **An MCP-compatible AI agent** - Claude Desktop, Cursor, or OpenCode
