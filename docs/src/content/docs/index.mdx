## What is Fizzy MCP?

Fizzy MCP is a **Model Context Protocol (MCP)** server that connects AI assistants like Claude, Cursor, and OpenCode to [Fizzy](https://app.fizzy.do) - the AI-powered task management platform.

With Fizzy MCP, your AI can:

- Create, update, and manage cards and boards
- Triage work into workflow columns
- Assign tasks to team members
- Track progress with tags and golden cards
- Collaborate through comments

## Quick Install

```bash
npx fizzy-do-mcp auth
```

This will:
1. Prompt for your Fizzy Personal Access Token
2. Validate the token with the Fizzy API
3. Offer to auto-configure detected AI agents

## Example Usage

Once configured, your AI assistant can interact with Fizzy naturally:

> "Create a new card on the Engineering board titled 'Implement user authentication'"

> "Show me all cards assigned to me that are marked as golden"

> "Move card #42 to the 'In Progress' column"

> "Add a comment to card #123 with my status update"

## Architecture

```
┌─────────────────┐     ┌─────────────────┐     ┌─────────────────┐
│   AI Agent      │────▶│   Fizzy MCP     │────▶│   Fizzy API     │
│ (Claude, etc.)  │◀────│    Server       │◀────│ (app.fizzy.do)  │
└─────────────────┘     └─────────────────┘     └─────────────────┘
        │                       │
        │                       │
        ▼                       ▼
   MCP Protocol          TypeScript Client
   (JSON-RPC)            (Result types)
```
