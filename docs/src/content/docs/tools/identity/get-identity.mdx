# fizzy_get_identity

Get the current user identity and list of accessible Fizzy accounts.

## Description

Use this tool to discover available accounts and verify authentication. It returns information about the authenticated user and all accounts they have access to.

## Parameters

This tool takes no parameters.

## Returns

```typescript
{
  accounts: Array<{
    id: string;
    slug: string;
    name: string;
    cards_count: number;
    auto_postpone_period_in_days: number | null;
    user: {
      id: string;
      name: string;
      email_address: string;
      avatar_url: string | null;
    };
  }>;
}
```

**Request:**
```
fizzy_get_identity()
```

- Verify authentication is working
- List available accounts to switch between
- Get user information for display
- Check card counts across accounts
