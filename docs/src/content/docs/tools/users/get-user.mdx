# fizzy_get_user
Get details of a specific user by ID.
## Parameters
| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `user_id` | `string` | Yes | The user ID |
## Example
```
fizzy_get_user({ user_id: "user_abc123" })
```
