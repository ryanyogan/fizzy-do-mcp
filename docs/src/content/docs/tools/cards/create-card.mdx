# fizzy_create_card

Create a new Fizzy card on a board.

## Description

Creates a new card with the specified title and optional description, tags, and status.

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `board_id` | `string` | Yes | The board ID to create the card on |
| `title` | `string` | Yes | The card title (min 1 character) |
| `description` | `string` | No | Card description (supports Markdown or HTML) |
| `status` | `string` | No | `drafted` or `published` (default: `published`) |
| `tag_ids` | `string[]` | No | Tag IDs to apply to the card |

### Status Values

| Value | Description |
|-------|-------------|
| `drafted` | Only visible to the creator |
| `published` | Visible to all board members |

## Returns

```typescript
{
  id: string;
  number: number;
  title: string;
  description: string | null;
  status: 'drafted' | 'published';
  board: { id: string; name: string };
  // ... other card fields
}
```

## Examples

### Simple card

```
fizzy_create_card({
  board_id: "board_abc123",
  title: "Implement user authentication"
})
```

### Card with Markdown description

```
fizzy_create_card({
  board_id: "board_abc123",
  title: "Fix login bug",
  description: "Users report issues on **mobile Safari**:\n\n- Can't submit form\n- Session expires"
})
```

### Card with tags

```
fizzy_create_card({
  board_id: "board_abc123",
  title: "Urgent: Server down",
  tag_ids: ["tag_urgent", "tag_ops"]
})
```

### Draft card

```
fizzy_create_card({
  board_id: "board_abc123",
  title: "WIP: New feature idea",
  status: "drafted"
})
```

## Markdown in Descriptions

Descriptions support Markdown formatting, which is automatically converted to HTML:

```markdown
## Summary
Brief description of the issue.

## Steps to Reproduce
1. Go to login page
2. Enter credentials
3. Click submit

## Expected vs Actual
**Expected:** Login success
**Actual:** Error message
```

### Supported Markdown Features

| Markdown | Example | Rendered |
|----------|---------|----------|
| Bold | `**text**` | **text** |
| Italic | `_text_` | _text_ |
| Links | `[text](url)` | Link |
| Lists | `- item` or `1. item` | Bullet/numbered lists |
| Headings | `## Heading` | Section headers |
| Code | `` `code` `` | Inline code |
| Blockquotes | `> quote` | Quoted text |

You can also use raw HTML if needed:

```html
<p>This is <strong>bold</strong> and <em>italic</em>.</p>
```
