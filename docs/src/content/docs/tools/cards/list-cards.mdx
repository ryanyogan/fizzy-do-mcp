# fizzy_list_cards

List Fizzy cards with optional filters. Use this to search and browse cards.

## Description

Returns cards matching the specified filters. This is one of the most frequently used tools for browsing and searching work items.

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `board_ids` | `string[]` | No | Filter by board ID(s) |
| `tag_ids` | `string[]` | No | Filter by tag ID(s) |
| `assignee_ids` | `string[]` | No | Filter by assignee user ID(s) |
| `indexed_by` | `string` | No | Filter by card state |
| `sorted_by` | `string` | No | Sort order |
| `terms` | `string[]` | No | Search terms |

### `indexed_by` Values

| Value | Description |
|-------|-------------|
| `all` | All cards |
| `closed` | Completed cards |
| `not_now` | Postponed cards |
| `stalled` | Inactive cards needing attention |
| `postponing_soon` | Cards about to auto-postpone |
| `golden` | Important/starred cards |

### `sorted_by` Values

| Value | Description |
|-------|-------------|
| `latest` | Most recently updated |
| `newest` | Most recently created |
| `oldest` | Oldest first |

## Returns

```typescript
Array<{
  id: string;
  number: number;
  title: string;
  description: string | null;
  status: 'drafted' | 'published';
  closed: boolean;
  postponed: boolean;
  golden: boolean;
  pinned: boolean;
  watching: boolean;
  board: { id: string; name: string };
  column: { id: string; name: string } | null;
  tags: Array<{ id: string; title: string }>;
  assignees: Array<{ id: string; name: string }>;
  created_at: string;
  updated_at: string;
}>
```

## Examples

### List all cards on a board

```
fizzy_list_cards({
  board_ids: ["board_abc123"]
})
```

### Find golden cards

```
fizzy_list_cards({
  indexed_by: "golden"
})
```

### Search by terms

```
fizzy_list_cards({
  terms: ["login", "bug"]
})
```

### Cards assigned to a user

```
fizzy_list_cards({
  assignee_ids: ["user_xyz789"]
})
```

### Combine filters

```
fizzy_list_cards({
  board_ids: ["board_abc123"],
  tag_ids: ["tag_urgent"],
  sorted_by: "newest"
})
```
