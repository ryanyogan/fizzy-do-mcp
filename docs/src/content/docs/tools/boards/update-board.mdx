# fizzy_update_board
Update an existing Fizzy board. Only board administrators can update boards.
## Parameters
| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `board_id` | `string` | Yes | The board ID |
| `name` | `string` | No | New board name |
| `all_access` | `boolean` | No | Access control |
| `auto_postpone_period_in_days` | `number` | No | Auto-postpone days |
