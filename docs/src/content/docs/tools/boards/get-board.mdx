# fizzy_get_board
Get details of a specific Fizzy board by its ID.
## Parameters
| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `board_id` | `string` | Yes | The unique identifier of the board |
## Example
```
fizzy_get_board({ board_id: "board_abc123" })
```
